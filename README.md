## Background
大一时，刚学完c语言，受一个高中同学之托，本着锻炼课内所学知识的想法，完成了这个学生成绩管理系统，虽然是控制台应用，但在操作方面我也尽可能地写得优雅和人性化，可能那会的代码风格不太规范，所以大家如果有宝贵意见，请千万在issue里写出来，我一定学习改进。如果有需要的同学欢迎自取借鉴，但初学者可不要照搬照抄哦，还是自己手动敲一点比较好。
